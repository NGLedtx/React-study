# 💰 Costs

A collection of financial web applications built with React, focused on learning, clean code, and real-world project simulation.

> This project is for educational purposes only.

## 🚀 Tech Stack
- React.js
- JavaScript (ES6+)
- Tailwind CSS
- React Router

## 📊 Features
- Financial dashboard
- Expense tracking
- Income management
- Full project management (CRUD operations)
